# SimpleSetup

SimpleSetup is moving to a dedicated terminal-first onboarding experience.

## Run the OpenClaw TUI

```bash
npm install
npm run tui:openclaw
```

Direct launch also works:

```bash
cd cli
node index.ts --tool openclaw
```

You can also start the tool picker:

```bash
npm run tui
```

The TUI guides you through OpenClaw configuration and generates a ready-to-run command.

## Run the website

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
